# Dev-Axioms 🚀

A modern developer knowledge platform for publishing technical documentation, blogs, and tutorials. The platform combines a powerful MDX documentation system with a developer-friendly blogging environment.

---

## 🌟 Features

* 📚 MDX-based documentation system
* ✍️ Developer blogging platform
* 📝 Built-in Markdown editor
* 👤 Developer profiles
* 🛠 Admin dashboard for content management
* ⚡ Fast and scalable architecture

---

## 🧑‍💻 Tech Stack

* **Frontend:** Next.js, React
* **Language:** TypeScript
* **Documentation:** MDX
* **Styling:** Tailwind CSS
* **Package Manager:** npm / pnpm

## 🚀 Getting Started

### 1️⃣ Clone the repository

```
git clone https://github.com/uk2610/dev-axioms.git
```

### 2️⃣ Navigate to the project folder

```
cd dev-axioms
```

### 3️⃣ Install dependencies

```
npm install --legacy-peer-deps
```

### 4️⃣ Run the development server

```
npm run dev
```

Open in browser:

```
http://localhost:3000
```
